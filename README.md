# DragMaster (크롬 확장 프로그램)

마우스 우클릭, 드래그(텍스트 선택), 그리고 복사 기능이 차단된 웹사이트에서 해당 제한을 강제로 해제해주는 빠르고 가벼운 크롬 브라우저 확장 프로그램입니다.

## 🌟 주요 기능

- **우클릭 차단 해제**: JavaScript 이벤트를 통해 막혀있는 컨텍스트 메뉴(`contextmenu`)를 정상적으로 호출합니다.
- **드래그 및 텍스트 선택 허용**: CSS의 `user-select: none` 속성을 무력화시키고, JavaScript의 `selectstart` 이벤트 차단을 우회하여 자유로운 텍스트 드래그(Highlight)를 돕습니다.
- **복사 방지 해제**: `copy`, `cut`, `paste` 단축키와 동작을 가로채어 정상 동작되도록 복원합니다.
- **블랙리스트 기반 자동 비활성화**: 구글(Google), 유튜브(YouTube), 페이스북(Facebook) 등 자체적인 고유 우클릭 메뉴를 보유한 복잡한 웹 서비스에서는 충돌 방지를 위해 자동으로 기능이 꺼집니다(Blacklist 예외 처리).

## 🛠 아키텍처 및 구현 방식 (Single Responsibility Principle)

기능별 역할을 분리하여 단일 책임 원칙(SRP)을 따르는 독립된 매니저(Manager) 클래스로 고도화되었습니다.

*   `main.js`: 확장 프로그램의 진입점(Entry Point). 도메인을 검사하여 확장을 실행할지 결정하고 각 모듈을 초기화합니다.
*   `BlacklistManager.js`: 정규식을 사용해 현재 사이트가 예외 처리되어야 할 도메인(Blacklist)인지 판별하는 역할을 수행합니다.
*   `StyleManager.js`: `<head>` 영역에 CSS 스타일(`user-select: text !important`) 인젝션을 수행하여 스타일 기반의 드래그 차단을 무력화합니다.
*   `EventManager.js`: 사이트 내 지정된 인라인 스크립트 이벤트를 지우고(`null`), 이벤트 리스너 캡처링(Capturing) 단계를 활용해 우클릭 및 복사 이벤트 전파를 차단(`e.stopPropagation()`)하여 보호 설정을 무력화합니다.

## 💡 로컬 설치 및 테스트 방법

이 확장 프로그램은 **Manifest V3** 규격을 기반으로 작성되었으며, 크롬 스토어 등록 전 로컬 환경에서 다음과 같이 임시 설치 및 테스트해 볼 수 있습니다.

1.  의 소스 코드를 다운로드하거나 폴더를 만듭니다 (`DragMaster` 등).
2.  크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력합니다.
3.  우측 상단의 **`개발자 모드`**를 켭니다.
4.  좌측 상단의 **`압축해제된 확장 프로그램을 로드합니다`** 버튼을 클릭하고 해당 소스가 있는 폴더를 선택합니다.
5.  폴더 내에 동봉된 `test.html` 파일을 크롬 브라우저로 끌어다 놓습니다.
6.  우클릭 방지 문구가 적힌 데모 화면에서 마우스 오른쪽 클릭 및 드래그가 정상적으로 동작하는지 확인합니다.

---
**주의**: 본 확장 프로그램은 학습 및 편의 목적을 위해 개발되었으며, 웹사이트에서 제공하는 민감한 개인정보나 저작물을 무단으로 추출하는 용도로 사용해서는 안 됩니다.
